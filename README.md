# Laptopie
A laptop recommendation system using AI for faster and more accurate results than ever.

## Enviroment variables to be defined
- DATABASE_URL
- GEMINI_API_KEY
- NEXTAUTH_URL
- LAPTOPIE_URL
- NEXTAUTH_SECRET
- GITHUB_SECRET
- GITHUB_ID
- PUSHER_APP_ID
- NEXT_PUBLIC_PUSHER_KEY
- PUSHER_SECRET
- GOOGLE_CLIENT_ID
- GOOGLE_CLIENT_SECRET
- EMAIL_SERVER_USER
- EMAIL_SERVER_PASSWORD
- EMAIL_FROM
  
## Getting started
- `npm i -g pnpm`
- `pnpm i`
- `pnpx prisma generate`
- `pnpm run dev`
